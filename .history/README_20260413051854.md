# salon-system

サロン業務向けのFAQ管理システムです。
スタッフがスマートフォンから素早く業務FAQを検索・確認できるよう設計しています。
従来のスプレッドシート運用で発生していた課題を解消し、現場で即利用できる業務支援ツールを目指しました。

また、PC操作に不慣れなスタッフでも直感的に利用できることを前提としています。

## 機能

### 認証・ユーザー管理
- ログイン機能
- ユーザー一覧表示
- ユーザー登録
- ユーザー編集
- ユーザー有効／停止切り替え
- ユーザー論理削除

### FAQ管理
- FAQ一覧表示
- FAQ登録
- FAQ一括登録
- FAQ CSVインポート
- FAQ編集
- FAQ表示／非表示切り替え
- FAQ論理削除
- FAQ履歴機能

### カテゴリ管理
- カテゴリ一覧表示
- カテゴリ登録
- カテゴリ一括登録
- カテゴリ編集
- カテゴリ論理削除

### その他
- 検索機能
- ページネーション
- 表示順管理（ドラッグ＆ドロップ）

## 補足

- FAQは物理削除を行わず、`deleted_at` による論理削除で管理します
- 表示／非表示は `is_visible` フラグで制御します
- FAQ更新時には履歴として更新前の内容を自動保存します
