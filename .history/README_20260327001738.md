# salon-system

サロン業務で使用するQ&A管理システムです。
スプレッドシートでの管理課題を解決するために作成しています。

## 機能
- Q&A一覧表示
- Q&A登録・編集
- Q&A一括登録・一括編集
- Q&A非表示
- 検索機能
- ページネーション
- 表示順管理（ドラッグ＆ドロップ）

## 使用技術
- Laravel
- MySQL
- Docker（Laravel Sail）
- phpMyAdmin
