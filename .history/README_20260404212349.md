# salon-system

スタッフが業務中や自宅からスマートフォンで素早く業務FAQを検索・確認できるようにするためのシステムです。  
従来のスプレッドシート管理で発生していた「運用のしづらさ」を解消し、現場で即使える業務支援ツールとして作成しています。  
また、PC操作に不慣れなスタッフでも直感的に利用できることを前提に設計しています。

## 機能
- FAQ一覧表示
- FAQ登録・編集
- FAQ一括登録・一括編集
- FAQ非表示
- FAQ履歴機能（更新前の内容を自動保存）
- 検索機能
- ページネーション
- 表示順管理（ドラッグ＆ドロップ）

## 使用技術
- Laravel
- MySQL
- Docker（Laravel Sail）
- phpMyAdmin
