# salon-system

サロン業務で使用するFAQ管理システムです。
スプレッドシートでの管理課題を解決するために作成しています。

## 機能
- FAQ一覧表示
- FAQ登録・編集
- FAQ一括登録・一括編集
- FAQ非表示
- 検索機能
- ページネーション
- 表示順管理（ドラッグ＆ドロップ）

## 使用技術
- Laravel
- MySQL
- Docker（Laravel Sail）
- phpMyAdmin
